Exploring all the component tooling available for blocks

== Description ==

This is a plugin for exploring and testing all the component tooling available for blocks.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/component-kitchen-sink` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= Is this plugin for a production website? =

No. It is for testing and exploring only.
